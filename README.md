## 💡 Back API
- branch 별 기능구분
  - master : 통합
  - signup : 회원가입
  - login : 로그인
  - checkposts : 전체 게시글 조회
  - registPost : 게시글 등록
  - checkpost : 게시글 조회
  - updatepost : 게시글 수정
  - deletepost : 게시글 삭제
  - registcomment : 댓글 생성
  - update_delete_checkcomment : 게시글과 댓글조회, 댓글 수정/삭제
